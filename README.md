# Top 500 _________

Automatically create a video for any image search term in the style of [top 5000 cheese.](https://youtu.be/N1GKJwmG5Aw)

## Development

## Prerequisites

- [Bun](https://bun.sh/)

### Setup

1. Clone the repository
2. Copy `.env.example` to `.env` and fill in the values
3. Add music to `public/audio`
4. Run `bun install` to install dependencies
5. Run `bun dev` to start remotion studio
6. Select "Top500Blank" on the left edit the props in the right panel.
